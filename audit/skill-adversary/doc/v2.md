# V2 features

- **Output-fuzzer**: agent that generates edge-case inputs (empty files, malformed YAML, incomplete frontmatter) and verifies that the target skill doesn't break
- **Task-observer**: integration with skill-creator to automatically launch skill-adversary after each skill-creator iteration (builder/critic loop)
- **Automatic hooks**: PostToolUse hook on skill-creator that triggers an adversary review after writing a SKILL.md
